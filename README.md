# 🤖 Customer Support AI System

## 📌 Project Overview
The Customer Support AI System is an AI-powered web application designed to automate customer interactions. It uses machine learning to understand user queries and provide intelligent responses in real-time.

## 🚀 Features
- AI-based chatbot for customer queries
- Machine Learning model for response prediction
- Web-based interface (HTML, CSS)
- Fast and real-time responses
- Training module for improving accuracy

## 🛠️ Tech Stack
- Backend: Python
- Frontend: HTML, CSS
- Machine Learning: NLP / Scikit-learn
- Database: SQLite (production_chat.db)
- Framework: Flask

## 📁 Project Structure
CUSTOMERSUPPORTAI/
│── data/                (Training data)
│── models/              (Saved ML models)
│── static/              (CSS, JS files)
│── templates/           (HTML files)
│   └── index.html
│── venv/                (Virtual environment)
│── ai_engine.py         (AI logic)
│── main.py              (Main application)
│── run.py               (Entry point)
│── train.py             (Model training script)
│── production_chat.db   (Database)
│── README.md            (Documentation)

## ⚙️ Installation & Setup
1. Clone Repository
cd CUSTOMERSUPPORTAI

2. Create Virtual Environment
python -m venv venv

3. Activate Environment
Windows: venv\Scripts\activate
Mac/Linux: source venv/bin/activate

4. Install Dependencies
pip install -r requirements.txt

## 🧠 Train Model
python train.py

## ▶️ Run Application
python run.py

Open browser:
http://127.0.0.1:5000
